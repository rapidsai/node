# Spatial Demo
A geospatial demo using cuSpatial and deck.gl.

## Featured Dependencies
- @rapidsai/cudf
- @rapidsai/cuspatial
- @rapidsai/jsdom
- @rapidsai/deckgl
- @rapidsai/glfw

## Data Requirements
Data required from `/data/263_tracts.arrow` and census data (auto downloaded) such as `200901.cny.gz`.

## Start
Example of starting demo:
```bash
yarn start
```
